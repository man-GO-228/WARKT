# Вывод спутника на орбиту луны
Команда:
Ожигов Тимофей Кириллович – тимлид, физик, математик
Пономарев Максим Сергеивич – математик, программист
Чамара Роман Романович – технический документалист, технический специалист
Мао Геэр – программист KSP, математик

Цель проекта
Разработка и реализация модели полёта с выводом спутника на стабильную орбиту вокруг Муны (в KSP — аналог «Луны») с использованием ракеты-носителя, спроектированной в симуляторе, и проведение расчётов характеристик полёта.

Задачи проекта
1. Изучение основ орбитальной механики для перелёта с орбиты Земли (Кербина) к Луне (Муне).
2. Разработка математической модели полёта, включая:
- Расчёт переходной траектории Гомана;
- Манёвры коррекции и захвата на лунную орбиту.
3. Проектирование и сборка ракеты-носителя в KSP с учётом требований миссии.
4. Проведение расчётов характеристик ступеней ракеты (Δv, массы, удельные импульсы).
5. Моделирование полёта в KSP и сравнение результатов с расчётными данными.
